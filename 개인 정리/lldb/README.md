  
  우선 디버깅 하려는 파일을 **컴파일** 할 때 **-g** 옵션을 줘야 디버깅 가능하다.   
  해당 파일을 컴파일 할 때 필요한 **오브젝트 파일**을 만들 때도 **-g** 줘야 한다.  
     
  해당 옵션을 주지 않으면 lldb로 프로세르를 읽을 때 어셈블리로 읽게 되는데, 이러면 코드를 보는 것 보다 가독성이 비교적 떨어진다.   
    
  </br>
